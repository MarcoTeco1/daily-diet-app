# 🍽️ Daily Diet

Aplicativo mobile desenvolvido com **React Native** que permite registrar e acompanhar refeições diárias, auxiliando no controle de uma dieta de forma simples, prática e visual.

---

## 📱 Sobre o projeto

O **Daily Diet** foi criado com o objetivo de ajudar o usuário a manter o controle alimentar, registrando refeições e acompanhando seu progresso por meio de estatísticas e feedback visual.

---

## 🚀 Funcionalidades

- Cadastro de refeições (nome, descrição, data e hora)
- Edição de refeições
- Exclusão com confirmação
- Visualização de estatísticas da dieta
- Filtro de refeições (todas, dentro ou fora da dieta)
- Armazenamento local com AsyncStorage
- Feedback visual após cadastro (dentro ou fora da dieta)

---

## 📊 Estatísticas disponíveis

- Percentual de refeições dentro da dieta
- Total de refeições cadastradas
- Quantidade de refeições dentro e fora da dieta
- Melhor sequência de refeições dentro da dieta

---

## 🛠️ Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- Styled-components
- React Navigation
- AsyncStorage (@react-native-async-storage/async-storage)
